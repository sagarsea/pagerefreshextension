No configurations are required for this extension. Please follow the below steps to ensure that the plugin functions correctly -

Step 1 Data Elements : Navigate to the 'Data Elements' tab and click 'Add Data Element'. In the configuration screen select extension - Acton PageRefresh & data element type - 'PageRefresh Detected'

Step 2 Rules : Navigate to your existing rule with Adobe Analytics & Adobe Target. Add action - 'Measure' to report into Adobe Analytics as context data & 'Persoanlize' to send an mBox Parameter to Adobe Target
